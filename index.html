Getting Started

  • To get started sign up for the developer program at http://developer.buildfire.com.
  • Once you've done so you should be issued an App ID that you can use test your plugin.
  • Open scripts/appContext.js and change the appId in window.appContext.currentApp .
  • Now you can begin developing in plugins/myPlugin.
  • if you choose to change that folder name make sure you change it as well in the scripts/appContext.js file
  • makes sure you run your local environment as we web application environment and not static files from teh file system. You will run into many issues this way
